# Session 007 - 首页报告页登录页绿色中医风改版

## 做了什么
- 统一首页、报告页、登录页的主视觉绿色语言，增强中医草本感。
- 计划把首页三诊评分条升级为接近报告页的浅绿渐变进度条，并调深首页 CTA 与登录按钮。
- 计划将首页与报告页 Hero 区域统一成从上到下白色过渡到浅绿色的竖向渐变。
- 已将首页 Hero 改为白 → 浅草本绿竖向渐变，并把首页三诊评分条改成更接近报告页的浅绿渐变进度条。
- 已将报告页 Hero 区域同步改为白 → 绿渐变，并把登录页主按钮加深为更稳定的墨绿 → 草本绿渐变。
- 根据返修反馈，再次增强了首页三诊评分条的可见度：提高轨道对比、增加条高、补充轻阴影，让它更明确地呈现为进度条。
- 同时把首页 CTA 与登录页主按钮的绿色收浅，避免主按钮过重、过闷。

## 变更前快照
- 首页三诊评分条虽为绿色进度条，但质感和报告页三诊评分语言不够统一。
- 首页“开始全套智能检测”和登录按钮都偏灰绿，草本感不够明确。
- 首页与报告页 Hero 绿色区域更像纯色底，不是白到绿的层次过渡。

## 涉及文件
- `lib/features/home/presentation/pages/home_page.dart`
- `lib/features/report/presentation/pages/report_page.dart`
- `lib/features/auth/presentation/pages/login_page.dart`

## 问题与决策
- 决策：不引入新主题层，直接在现有页面内统一关键颜色和渐变，控制改动范围。
- 决策：首页三诊评分条优先向报告页“三诊评分”靠拢，但仍保留首页紧凑布局。
- 决策：按钮与 Hero 渐变统一采用“近白 → 草本浅绿 → 墨绿强调”的中医风配色。

## 验证结果
- `lsp_diagnostics`：`home_page.dart`、`report_page.dart`、`login_page.dart` 均无错误。
- `flutter analyze lib/features/home/presentation/pages/home_page.dart lib/features/report/presentation/pages/report_page.dart lib/features/auth/presentation/pages/login_page.dart`：无新增错误；剩余为项目既有 `unused_element` / `unnecessary_underscores` 信息与警告，不属于本次改动引入。
- 返修后再次执行 `flutter analyze lib/features/home/presentation/pages/home_page.dart lib/features/auth/presentation/pages/login_page.dart`：无新增错误；仍仅剩项目既有 `unused_element` / `unnecessary_underscores`。

## 下次计划

## 遗留风险
- 当前为页面内局部统一，尚未抽成全局 design token，后续新增页面仍可能出现色值漂移。
- 视觉效果已在静态层面校验，但 Hero 顶部白绿过渡和按钮阴影层次仍建议真机看一次，确认在不同亮度屏幕下不过灰也不过艳。
