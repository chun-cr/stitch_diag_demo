# Session 006 - 历史报告页折线图改版

## 做了什么
- 重新设计历史报告页面的健康走势与风险指数折线图，重点优化移动端 X 轴标签、线条观感与触摸交互。
- 目标是移除倾斜日期、默认常显数据点和偏后台风的图表语言，改成更轻、更顺滑的移动端趋势图。
- 将两张图统一为“头 / 中 / 尾”三个水平日期标签，默认隐藏数据点，仅在触摸时显示指示点与浮窗。
- 为健康走势与风险指数走势图补充统一的轻量网格、底边线与柔和 tooltip 样式，保持现有页面配色不变。

## 变更前快照
- X 轴日期按隔点显示，但使用了倾斜排版，手机端阅读负担重。
- 健康走势和风险指数两张图默认都显示所有数据点，画面颗粒感偏强。
- 图表未配置触摸态专属高亮和信息浮窗，趋势浏览与精确查看没有分层。

## 涉及文件
- `lib/features/history/presentation/pages/history_page.dart`

## 问题与决策
- 决策：X 轴改为只显示“头 / 中 / 尾”三个水平日期标签，保留趋势感，不挤压移动端空间。
- 决策：默认隐藏所有折线圆点，仅在触摸时通过 touched indicator 显示当前位置圆点与引导线。
- 决策：保持现有 `fl_chart` 方案，不新增依赖，只通过现有 API 完成高端化改版。
