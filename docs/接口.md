}



## 注册challenge

注册challenge的url:/api/v1/saas/mobile/auth/register/verification-code/challenge
请求参数：
{
    "countryCode": "string",
    "phoneNumber": "string"
}
返回响应格式：
{
    "code": 0,
    "message": "string",
    "messageKey": "string",
    "requestId": "string",
    "timestamp": "2019-08-24T14:15:22.123Z",
    "data": {
        "challengeId": "string",
        "captchaRequired": true,
        "captcha": {
            "provider": "string",
            "payload": {}
        },
        "expireAt": "string"
    }
}
