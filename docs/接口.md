# 注册



## 密码注册

/api/v1/saas/mobile/auth/register/password
注册url
参数（必须）
{
    "countryCode": "string",
    "phoneNumber": "string",
    "password": "string"
}



## 注册challenge

注册challenge的url:/api/v1/saas/mobile/auth/register/verification-code/challenge
请求参数：
{
    "countryCode": "string",
    "phoneNumber": "string"
}
返回响应格式：
{
    "code": 0,
    "message": "string",
    "messageKey": "string",
    "requestId": "string",
    "timestamp": "2019-08-24T14:15:22.123Z",
    "data": {
        "challengeId": "string",
        "captchaRequired": true,
        "captcha": {
            "provider": "string",
            "payload": {}
        },
        "expireAt": "string"
    }
}

## 注册如需人机验证

从上一步拿到challengeId、captchaProvider、captchaPayload

url：/api/v1/saas/mobile/auth/verification-code/captcha/verify

请求体:

{
    "challengeId": "string",
    "captchaProvider": "string",
    "captchaPayload": {}
}

返回响应：

{
    "code": 0,
    "message": "string",
    "messageKey": "string",
    "requestId": "string",
    "timestamp": "2019-08-24T14:15:22.123Z",
    "data": {
        "challengeId": "string",
        "captchaVerified": true
    }
}

## 使用手机号验证码注册

用户得到验证码verificationCode

url：/api/v1/saas/mobile/auth/register/verification-code

请求体：

{
    "challengeId": "string",
    "verificationCode": "string",
    "inviteTicket": "string"（可选）
}

返回响应：

{
    "code": 0,
    "message": "string",
    "messageKey": "string",
    "requestId": "string",
    "timestamp": "2019-08-24T14:15:22.123Z",
    "data": {
        "accessToken": "string",
        "refreshToken": "string",
        "tokenType": "string",
        "expiresIn": 0,
        "scope": "string"
    }
}

# 登录

## 账号密码登录

url：/api/v1/saas/mobile/auth/login/password

请求体：

{
    "loginType": "string",（登录标识类型，前端可不传；后端会忽略该字段并自行识别）
    "loginValue": "string",
    "countryCode": "string",
    "password": "string",
    "inviteTicket": "string"（可选）
}

返回响应：

{
    "code": 0,
    "message": "string",
    "messageKey": "string",
    "requestId": "string",

    "data": {
        "accessToken": "string",
        "refreshToken": "string",
        "tokenType": "string",
        "expiresIn": 0,
        "scope": "string"
    }
}
